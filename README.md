# Quiz-Bot

A simple Python project demonstrating a basic CLI function and utility function.  
Built with **Python** and managed with **uv** for dependencies and virtual environments.  

---

## Features

- CLI function: `main()` prints a welcome message.
- Utility function: `hello()` returns a greeting string.
- Unit testing with **pytest**.
- Linting ready with **Ruff**.
- Virtual environment and dependency management with **uv**.
